Acerca de Este Proyecto

Bienvenido a la página de información sobre nuestro juego del Ahorcado. Este proyecto fue desarrollado como parte de una práctica grupal para aprender y aplicar conceptos de desarrollo web colaborativo utilizando herramientas como Git, GitHub y tecnologías básicas como HTML, CSS y JavaScript.

Objetivo del Proyecto

El objetivo principal de este proyecto es proporcionar una experiencia interactiva y educativa para los usuarios, mientras practicamos habilidades clave en el desarrollo web, como:

  • Diseño responsivo y estilizado con CSS.
  • Interactividad y lógica básica con JavaScript.
  • Colaboración en equipo utilizando Git y GitHub.
  • Implementación de buenas prácticas en diseño y código limpio.

Características del Juego

Nuestro juego del Ahorcado tiene las siguientes características:

  • Modo oscuro: Diseño moderno y minimalista con una paleta de colores oscura.
  • Palabras comunes: Incluye palabras en español con pistas para ayudarte.
  • Interactividad: Envía letras presionando "Enter" sin necesidad de botones adicionales.
  • Animaciones y SVG: El ahorcado se dibuja dinámicamente con gráficos vectoriales.
  • Responsivo: Se adapta perfectamente a dispositivos móviles y pantallas grandes.
